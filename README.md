# Mod_ADC
O mod_adc é uma biblioteca modular desenvolvida em C para o microcontrolador ATmega328P, destinada a facilitar a configuração e o uso do conversor analógico-digital (ADC) de forma simples, clara e reutilizável.  O módulo abstrai a manipulação direta dos registradores (ADMUX, ADCSRA, ADCSRB, DIDR0)
